# OmniCore V3 — Streamlit Direct

Versión de la Session 3 pasada a una sola app de Streamlit, lista para deploy sin backend aparte.

## Qué hace
- Crea proyectos
- Genera un juego base en Pygame (`car_runner`)
- Organiza archivos
- Prepara deploy
- Deja descargar cada proyecto en ZIP

## Correr local
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy en Streamlit Cloud
1. Sube esta carpeta a GitHub.
2. En Streamlit Cloud crea una app nueva.
3. Selecciona el repo.
4. App file: `app.py`
5. Python 3.11 recomendado.

## Nota
El juego generado es descargable como proyecto; Streamlit sirve aquí como panel de creación y exportación.
